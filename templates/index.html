Information

Cet outil vous permet de générer des URLs d'API Grist avec filtres pour intégrer vos données dans le champ référentiel avancé de Démarche Numérique. La colonne choisie comme filtre principal correspondra à la valeur saisie par l'usager dans DN.

Générez votre token dans les paramètres de votre compte Grist
Trouvez l'ID dans l'URL de votre document Grist : https://grist.../docs/ID_DU_DOCUMENT/...
Cochez pour combiner plusieurs colonnes avec des conditions exactes, "contient" (insensible à la casse) ou listes de valeurs

Ce mode génère un filtre exact (la colonne choisie doit correspondre exactement à la valeur saisie par l'usager). Pour une recherche partielle ("contient", "commence par") ou insensible à la casse, activez les conditions SQL avancées ci-dessus.

Conditions SQL

Ajoutez plusieurs conditions. La première condition (encadrée en bleu) correspond au filtre principal : la valeur saisie par l'usager dans DN. Pour chaque condition, choisissez une colonne, un opérateur (exact, contient, dans une liste) et soit un placeholder {id}, soit une valeur fixe. Toutes les conditions sont combinées avec le connecteur choisi ci-dessous.
